## エッジケースサンプル

### 空配列の例

```json
{
  "科目名": "サンプル科目",
  "開講情報一覧": [],
  "履修情報一覧": []
}
```

### null値の例

```json
{
  "科目名": null,
  "開講情報一覧": [
    {
      "年度": null,
      "シラバス一覧": [
        {
          "教科書一覧": null
        }
      ]
    }
  ],
  "履修情報一覧": null
}
```

### 複数教員の例

```json
{
  "科目名": "AI入門",
  "開講情報一覧": [
    {
      "年度": 2025,
      "シラバス一覧": [
        {
          "担当教員一覧": [
            { "氏名": "田中 太郎", "役割": "主担当" },
            { "氏名": "佐藤 花子", "役割": "副担当" }
          ]
        }
      ]
    }
  ]
}
```
